CSS Menu Icon


info

This is a demo showing how css can be used to produce a menu icon which can, if desired, be animated. Both a static and animated icon are shown. These icons are taken from information presented on w3schools.

The css styling for the page is almost entirely based on w3.css supplemented by a normal css file to provide styling for the file icons.

To close this pannel please left click the info icon again.

The menu icons shown on this page are made visible through the use of div sections with appropriate css styling to show three horizontal bars.

The icon on the left is a static icon while that on the right is animated with the help of JavaScript. Either one can be used as a way to open a file menu or to fire other events.

Left click the animated icon to see the effect.

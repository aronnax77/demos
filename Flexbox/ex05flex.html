Flexbox #5 - Notes regarding flex

The flex shorthand property

diplay: flex: none | [ <'flex-grow'> <'flex-shrink'>? || <'flex-basis'> ]

When flex is not specified for an item this is automatically given the values flex: 0 0 auto where the three parameters are related to the properties flex-grow, flex-shrink, and flex-basis

A demo has been provided here for flex-grow . 0 effectively limits the items in the flex container to their original dimensions.

flex: can be specified for an item with one, two, or three parameters. These are explained in the article flex on MDN's site.

flex-shrink defines the ability for a flex item to shrink if necessary while flex-basis defines the default size of an element before the remaining space is distributed.

A demonstration of flex-shrink can be found here. and a demo for flex-basis can be seen here.

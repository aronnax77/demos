Parent/Child communication using '$emit' and 'props' can look very similar in appearance on the rendered page. See this example of the use of '$emit'. 'props', demonstrated here, is used to recieve a message from the parent to the child while '$emit' is used to send a message from the child to the parent.

In this example the input field and the button are part of the parent. After a message is written into the input field the send message button calls a handler on the parent which sets the 'parentMessage' data field on the parent and finally clears the input field.

Transmission or sending of the message to the child now relies on a binding between 'message' the 'props' property on the child and the data held by the parent parentMessage'.
